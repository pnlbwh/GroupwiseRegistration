DiffeomorphicDemonsRegistrationFilter (inherits from PDEDeformableRegistrationFilter) creates a new demons registration
function pointer - in its constructor method.
  typename DemonsRegistrationFunctionType::Pointer drfp;
  drfp = DemonsRegistrationFunctionType::New();

This function is type of ESMDemonsRegistrationFunction  (DiffeomorphicDemonsRegistrationFilter's header)

ESMDemonsRegistrationFunction (Fast implementation of the symmetric demons registration force - Vercauteren) inherits from 
PDEDeformableRegistrationFunction.


DiffeomorphicDemonsRegistrationFilter is instantiated in DemonsRegistration.

DemonsRegistration
  -> DiffeomorphicDemonsRegistrationFilter :  PDEDeformableRegistrationFilter filter 
    -> ESMDemonsRegistrationFunction : PDEDeformableRegistrationFunction  drfp          
  -> MultiResolutionPDEDeformableRegistration2  MultiResRegistrationFilterType multires   
  -> multires->SetRegistrationFilter( filter );
